AOP 란(Aspect Oriented Programming) 관점지향 프로그램 스프링 어플리케이션은 대부분 특별한 경우를 제외 하고는 MVC 웹 어플리케이션에서는 Web Layer, Business Layer, Data Layer 로 정의가 되어있습니다. - Web Layer : REST Api를 제공하며, Client 중심의 로직 적용 - Business Layer : 내부 정책에 따른 logic 를 개발하며, 주로 해당 부분을 개발 - Data Layer : 데이텁 베이스 및 외부와의 연동을 처리
